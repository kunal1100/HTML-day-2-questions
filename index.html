HTML Interview questions day 2



1. Difference between HTML tags and elements

Tags = HTML tags are used to hold the HTML element. HTML tag starts with "<" and ends with ">"

Elements = HTML element holds the content. Whatever written within a HTML tag are HTML elements.



2. Explain br tags

The br tag inserts a single line break. The br tag is useful for writing addresses or poems. The br tag is an empty tag which means that it has no end tag.



3. Explain different types of lists in HTML

There are three list types in HTML:
unordered list — used to group a set of related items in no particular order
ordered list — used to group a set of related items in a specific order
description list — used to display name/value pairs such as terms and definitions



4. What are semantic tags

A semantic element clearly describes its meaning to both the browser and the developer.

Examples of semantic elements: form, table, and article - Clearly defines its content.
